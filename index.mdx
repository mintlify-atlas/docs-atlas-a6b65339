      ```json Claude Code (.mcp.json)
        "mcpServers": {
          "midos": {
            "url": "https://mcp.midos.dev/mcp"
          }
        }
      }
      ```

      ```json Cursor / Windsurf (MCP Settings)
      {
        "mcpServers": {
          "midos": {
            "url": "https://mcp.midos.dev/mcp",
            "transportType": "streamable-http"
          }
        }
      }
      ```
      </CodeGroup>
    </Step>

    <Step title="Call agent_handshake for personalized setup">
      Tell MidOS about your environment to get optimized results.

      ```bash
      agent_handshake(
        model="claude-opus-4-6",
        client="claude-code",
        languages="python,typescript",
        frameworks="fastapi,react"
      )
      ```
    </Step>

    <Step title="Search the knowledge base">
      Use semantic search to find battle-tested patterns and skills.

      ```bash
      semantic_search("React 19 Server Components patterns")
      ```
    </Step>
