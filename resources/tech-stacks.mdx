## How to Query by Stack

Use the `list_skills` tool with a stack filter:

```python
# List all React-related skills
list_skills(stack="react")

# List all Python backend skills
list_skills(stack="python")

# Search within a specific stack
search_knowledge("dependency injection", stack="fastapi")
```
