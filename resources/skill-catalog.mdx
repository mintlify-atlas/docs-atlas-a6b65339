MidOS includes **200+ skill packs** covering modern frameworks, languages, databases, infrastructure, and AI/ML tools. Each skill is battle-tested, curated, and organized by technology stack.

## What is a Skill Pack?

Skill packs are **organized, actionable knowledge** (Layer 2 in the knowledge pipeline) that provide:

- Production-tested patterns and best practices
- Stack-specific implementation guides
- Version-aware documentation (e.g., React 19, NestJS 11, Prisma 7)
- Cross-validated against the corpus
- Tagged with prerequisites and dependencies

## How to Access Skills

Use the `get_skill` tool via MCP:

```python
# Get a specific skill pack
get_skill("nextjs")

# List skills by stack
list_skills(stack="react")

# Search across all skills
search_knowledge("FastAPI dependency injection")
```
