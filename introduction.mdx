## What is MidOS?

MidOS is a curated knowledge API specifically designed for AI coding agents. Unlike traditional documentation scrapers that return raw docs, MidOS delivers **battle-tested patterns, validated discoveries, and empirically verified knowledge** across 20+ tech stacks.

Think of it as the difference between:
- **Raw docs services**: "Here's the React documentation from reactjs.org"
- **MidOS**: "Here's a battle-tested pattern for React 19 optimistic updates with useOptimistic + Server Actions, validated February 2026"

<Note>
MidOS is accessed via the [Model Context Protocol (MCP)](https://modelcontextprotocol.io), making it compatible with Claude Code, Cursor, Windsurf, Cline, and other MCP-enabled AI coding tools.
</Note>
