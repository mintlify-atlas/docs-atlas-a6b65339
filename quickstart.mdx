## Connect to MidOS

MidOS uses the Model Context Protocol (MCP) to integrate with AI coding tools. Choose your client below:

<Tabs>
  <Tab title="Claude Code">
    Add to `.mcp.json` or `~/.claude/settings.json`:

    ```json
    {
      "mcpServers": {
        "midos": {
          "url": "https://mcp.midos.dev/mcp"
        }
      }
    }
    ```

    <Note>
    Claude Code automatically connects to the MidOS cloud instance. No installation required.
    </Note>
  </Tab>

  <Tab title="Cursor">
    1. Open **Settings** → **MCP Settings**
    2. Click **Add Server**
    3. Configure:
       - **Name**: `midos`
       - **URL**: `https://mcp.midos.dev/mcp`
       - **Transport**: Streamable HTTP

    <Check>
    Cursor will connect on next reload.
    </Check>
  </Tab>

  <Tab title="Windsurf">
    1. Open **Settings** → **MCP Settings**
    2. Click **Add Server**
    3. Configure:
       - **Name**: `midos`
       - **URL**: `https://mcp.midos.dev/mcp`
       - **Transport**: Streamable HTTP

    <Check>
    Windsurf will connect immediately.
    </Check>
  </Tab>

  <Tab title="Cline">
    Add to your Cline MCP settings:

    ```json
    {
      "mcpServers": {
        "midos": {
          "url": "https://mcp.midos.dev/mcp",
          "transportType": "streamable-http"
        }
      }
    }
    ```
  </Tab>

  <Tab title="Claude Desktop">
    Add to `claude_desktop_config.json`:

    **macOS**: `~/Library/Application Support/Claude/claude_desktop_config.json`  
    **Windows**: `%APPDATA%\Claude\claude_desktop_config.json`

    ```json
    {
      "mcpServers": {
        "midos": {
          "url": "https://mcp.midos.dev/mcp"
        }
      }
    }
    ```

    <Warning>
    Claude Desktop requires a restart after config changes.
    </Warning>
  </Tab>
</Tabs>

## First Tool Call: Agent Handshake

After connecting, **always call `agent_handshake` first** to personalize your MidOS experience. This configures optimal settings for your specific model, CLI, and tech stack.

<CodeGroup>
```python Python/FastAPI Stack
agent_handshake(
    model="claude-opus-4-6",
    client="claude-code",
    languages="python,typescript",
    frameworks="fastapi,react"
)
```

```typescript TypeScript/Next.js Stack
agent_handshake(
    model="claude-sonnet-4-5",
    client="cursor",
    languages="typescript,python",
    frameworks="nextjs,tailwind"
)
```

```go Go/Kubernetes Stack
agent_handshake(
    model="claude-opus-4-6",
    client="windsurf",
    languages="go,python",
    frameworks="kubernetes,terraform"
)
```

```rust Rust Stack
agent_handshake(
    model="gemini-2.5-pro",
    client="cline",
    languages="rust,typescript",
    frameworks="actix,postgres"
)
```
</CodeGroup>

<Accordion title="What does agent_handshake return?">
The handshake returns a personalized configuration including:
- Recommended search strategies for your stack
- Context window optimization tips for your model
- Pre-filtered skill packs matching your languages and frameworks
- Stack-specific best practices

Example response:
```

Model: claude-opus-4-6 (200K context)
Client: claude-code
Stack: python, typescript, fastapi, react

Recommended Tools:
- semantic_search with stack='python,fastapi' for backend queries
- get_skill('fastapi-patterns-2026') for production patterns
- list_skills(stack='react') for frontend components

Your context window allows deep skill exploration.
Pro tip: Always search before building — MidOS has 46K+ chunks.
```
</Accordion>

## Second Tool Call: Search Knowledge

Now search for what you need. MidOS provides multiple search strategies:

<Tabs>
  <Tab title="Semantic Search (Recommended)">
    Use natural language queries with vector similarity search:

    ```python
    semantic_search(
        query="implement caching for API responses with Redis",
        top_k=5,
        stack="python,fastapi"
    )
    ```

    **Best for**: Conceptual queries, "how do I..." questions, finding patterns

    <Accordion title="Example Output">
    Returns ranked results with relevance scores. The top match includes production-ready code patterns for FastAPI + Redis caching with automatic invalidation and TTL management.
    </Accordion>
  </Tab>

  <Tab title="Keyword Search">
    Search for specific terms across the knowledge base:

    ```python
    search_knowledge(
        query="FastAPI dependency injection",
        max_results=5
    )
    ```

    **Best for**: Finding specific functions, error messages, or exact terminology
  </Tab>

  <Tab title="Smart Search">
    Automatic mode selection (keyword, semantic, or hybrid):

    ```python
    smart_search("PostgreSQL JSONB indexing strategies")
    ```

    **Best for**: When you're not sure which search strategy to use
  </Tab>
</Tabs>

## Browse Available Skills

List all skill packs, optionally filtered by your tech stack:

```python
# List all skills
list_skills()

# Filter by stack (prioritizes matching skills)
list_skills(stack="python,react")

# Text filter
list_skills(filter="postgres")
```

## Retrieve a Specific Skill

Get the full content of a skill pack:

```python
get_skill("FASTAPI_PRODUCTION_PATTERNS")
```

This returns a complete skill document with:
- Production-ready code examples
- Common pitfalls and how to avoid them
- Performance optimization tips
- Security best practices
- Testing strategies

```python
project_status()
```

The `project_status()` tool returns:
- Real-time knowledge base statistics
- Vector store health
- Available tools with examples
- Quick-start guide
- Pro tips for using MidOS effectively

## Using with API Key (Pro Tier)

To access Pro tier features, add your API key to the MCP configuration:

<Tabs>
  <Tab title="Claude Code / Claude Desktop">
    ```json
    {
      "mcpServers": {
        "midos": {
          "url": "https://mcp.midos.dev/mcp",
          "headers": {
            "Authorization": "Bearer midos_sk_your_key_here"
          }
        }
      }
    }
    ```
  </Tab>

  <Tab title="Cursor / Windsurf / Cline">
    Add the Authorization header in your MCP server configuration:
    
    **Header Name**: `Authorization`  
    **Header Value**: `Bearer midos_sk_your_key_here`
  </Tab>
</Tabs>
