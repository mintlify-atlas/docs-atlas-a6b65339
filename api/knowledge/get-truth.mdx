## Overview

The `get_truth` tool retrieves truth patch documents - verified corrections to common misconceptions in AI development, agent systems, and software engineering. Truth patches help avoid pitfalls and anti-patterns.

<Note>
  Full content of truth patches requires a **MidOS PRO API key**. Get your key at [midos.dev/pricing](https://midos.dev/pricing)
</Note>

## Function Signature

```python
async def get_truth(name: str) -> str
```

## Parameters

<ParamField path="name" type="string" required>
  The truth patch name to retrieve. Case-insensitive matching is supported.
  
  Examples:
  - `"AGENT_MITIGATIONS_CONTEXT_OVERFLOW"` - Context overflow mitigation
  - `"AGENT_MITIGATIONS_FABRICATION"` - Hallucination prevention
  - `"TRUTH_LLM_PLANNING_LIMITATION"` - LLM planning limitations
  - `"MCP_ECOSYSTEM_UPDATES_2026"` - MCP ecosystem status
</ParamField>

## Return Value

Returns a string containing:
- **Success**: Full content of the truth patch document in Markdown format
- **Not Found**: Error message with list of available truth patches (up to 20)

```
Truth patch not found: {name}

Available truth patches: ['AGENT_MITIGATIONS_CONTEXT_OVERFLOW', ...]
```

## Behavior

1. **Exact Match**: First attempts to find an exact match for `{name}.md`
2. **Case-Insensitive Search**: If no exact match, performs case-insensitive search
3. **Error with Suggestions**: Returns helpful error with available patch names
4. **Authentication**: Requires valid MidOS PRO API key for full content
