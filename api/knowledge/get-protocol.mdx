## Overview

The `get_protocol` tool retrieves protocol documents - formal specifications, standards, and communication protocols used in the MidOS ecosystem. Protocols define how systems, agents, and components interact.

## Function Signature

```python
async def get_protocol(name: str) -> str
```

## Parameters

<ParamField path="name" type="string" required>
  The protocol name to retrieve. Case-insensitive matching is supported.
  
  Examples:
  - `"PROTOCOL_NEURAL_LINK"` - Neural link communication protocol
  - `"PROTOCOL_AGENT_HANDSHAKE"` - Agent initialization protocol
  - `"MCP_PROTOCOL_SPEC"` - Model Context Protocol specification
  - `"A2A_PROTOCOL"` - Agent-to-Agent communication
</ParamField>

## Return Value

Returns a string containing:
- **Success**: Full content of the protocol document in Markdown format
- **Not Found**: Error message indicating the protocol was not found

```
Protocol not found: {name}
```

## Behavior

1. **Exact Match**: First attempts to find an exact match for `{name}.md`
2. **Case-Insensitive Search**: If no exact match, performs case-insensitive search
3. **Simple Error**: Returns basic error message if protocol doesn't exist

## Protocol Types

### Communication Protocols

Define how systems communicate:
- Agent-to-Agent (A2A) protocols
- Model Context Protocol (MCP) specifications
- Inter-process communication patterns
- API communication standards

### Behavioral Protocols

Define system behaviors:
- Agent initialization sequences
- Handshake procedures
- State transition rules
- Error handling protocols

### Data Protocols

Define data formats and structures:
- Message format specifications
- Knowledge representation standards
- Serialization protocols
- Schema definitions

## Usage Examples

### Get Agent Handshake Protocol

```python
result = await get_protocol(name="PROTOCOL_AGENT_HANDSHAKE")
# Returns the agent initialization and handshake specification
```

### Retrieve Neural Link Protocol

```python
result = await get_protocol(name="PROTOCOL_NEURAL_LINK")
# Returns the neural link communication protocol specification
```

### Get A2A Protocol Spec

```python
result = await get_protocol(name="A2A_PROTOCOL")
# Returns Agent-to-Agent communication protocol
```

### Case-Insensitive Lookup

```python
# These are equivalent:
result = await get_protocol(name="protocol_neural_link")
result = await get_protocol(name="PROTOCOL_NEURAL_LINK")
result = await get_protocol(name="Protocol_Neural_Link")
```

### Handle Missing Protocol

```python
result = await get_protocol(name="NONEXISTENT_PROTOCOL")
# Returns: "Protocol not found: NONEXISTENT_PROTOCOL"
```

## Protocol Documentation Structure

Most protocols follow this structure:

1. **Overview**: Purpose and scope
2. **Specification**: Formal protocol definition
3. **Message Formats**: Data structures and schemas
4. **Sequences**: Interaction sequences and flows
5. **Error Handling**: Error codes and recovery
6. **Examples**: Reference implementations
7. **Compliance**: Validation and testing
