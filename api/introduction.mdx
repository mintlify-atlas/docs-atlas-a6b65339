## Welcome to the MidOS API

MidOS exposes 68 MCP (Model Context Protocol) tools for AI agents to access curated developer knowledge. Every tool provides validated, production-tested patterns — not raw documentation dumps.

## MCP Endpoint

Connect your MCP client to:

```
https://mcp.midos.dev/mcp
```

All tools are available via this single endpoint using the MCP protocol over HTTP (streamable transport).

## Tool Categories

MidOS organizes tools into four functional categories:

### Search Tools (8 tools)

Query the knowledge base with keyword, semantic, or hybrid search strategies.

| Tool | Description | Tier |
|------|-------------|------|
| `search_knowledge` | Keyword search across 46,000+ curated chunks | Dev (free) |
| `semantic_search` | Vector search with Gemini embeddings (670K vectors) | Dev (free) |
| `hybrid_search` | Combined keyword + semantic with reranking | Dev (free) |
| `smart_search` | Unified search with auto/keyword/semantic/hybrid modes | Dev (free) |

### Knowledge Tools (11 tools)

Access structured knowledge layers: skills, EUREKA discoveries, truth patches, and protocols.

| Tool | Description | Tier |
|------|-------------|------|
| `list_skills` | Browse 200+ skill packs, filterable by tech stack | Dev (free) |
| `get_skill` | Retrieve complete skill pack by name | Dev (free) |
| `get_eureka` | Access 383 validated breakthrough discoveries | Dev (free) |
| `get_truth` | Retrieve empirically verified truth patches | Dev (free) |
| `get_protocol` | Get protocol and pattern documentation | Dev (free) |
| `get_aotc` | Ahead-of-the-curve frontier research (140 SOTA) | Pro ($20/mo) |

### Agent Tools (9 tools)

Personalization, memory, and learning capabilities for AI agents.

| Tool | Description | Tier |
|------|-------------|------|
| `agent_handshake` | Personalized onboarding for your model + stack | Dev (free) |
| `episodic_search` | Search agent session history and past experiences | Pro ($20/mo) |
| `episodic_store` | Store learnings for future agent sessions | Pro ($20/mo) |
| `chunk_code` | AST-based code chunking for better RAG retrieval | Dev (free) |
| `where_was_i` | Recover last session context instantly | Pro ($20/mo) |
| `create_plan` | Persistent execution plans for multi-step tasks | Pro ($20/mo) |

### System Tools (6 tools)

System status, health checks, and operational monitoring.

| Tool | Description | Tier |
|------|-------------|------|
| `hive_status` | System health and live statistics | Dev (free) |
| `project_status` | Live dashboard + quick-start guide | Dev (free) |
| `memory_stats` | Vector store and memory system statistics | Dev (free) |
| `pool_status` | Multi-instance coordination status | Dev (free) |
| `pool_signal` | Signal actions to coordination pool | Pro ($20/mo) |

## Access Tiers

MidOS uses a two-tier access model:

### Dev Tier (Free)

- No API key required
- 100 queries per month
- Access to 8 community tools + most dev tools
- Full knowledge base search (46K chunks)
- All skill packs, EUREKA discoveries, and truth patches
- Vector search with 670,000+ embeddings

### Pro Tier ($20/month)

- Requires API key
- 25,000 queries per month
- All dev tier tools + premium features
- AOTC (ahead-of-the-curve) frontier research
- Episodic memory (search + store)
- Security scanning tools
- Persistent execution plans
- Session recovery

See the [Authentication guide](/api/authentication) for details on obtaining and using API keys.

## How to Call Tools

### Via MCP Client Configuration

Configure your MCP client to connect to MidOS:

<CodeGroup>

```json Claude Code (.mcp.json)
{
  "mcpServers": {
    "midos": {
      "url": "https://mcp.midos.dev/mcp"
    }
  }
```

```json Cursor/Windsurf
  "mcpServers": {

      "url": "https://mcp.midos.dev/mcp",
      "transportType": "streamable-http"

```

```json Cline

  "mcpServers": {
    "midos": {
      "url": "https://mcp.midos.dev/mcp",
```

</CodeGroup>

### Example Tool Call

Once connected, your AI agent can invoke tools directly:

```
agent_handshake(
  model="claude-opus-4-6",
  client="claude-code",
  languages="python,typescript",
  frameworks="fastapi,react",
  project_goal="Building a real-time analytics dashboard"
)
```

```
semantic_search(
  query="optimistic updates in React 19",
  top_k=5,
  stack="react,typescript"
)
```

## Tool Signatures

All tools are defined in `modules/mcp_server/midos_mcp.py` using FastMCP decorators. Each tool includes:

- **Function signature**: Parameters with types and defaults
- **Docstring**: Description and parameter documentation
- **Return type**: String output (formatted markdown or JSON)
- **Error handling**: `ToolError` exceptions for invalid input

## Rate Limiting

All requests are subject to rate limits based on your tier:

- **Dev tier**: 100 queries/month
- **Pro tier**: 25,000 queries/month

Rate limits are tracked per API key (authenticated) or per IP address hash (anonymous). Limits reset on the first day of each month.

See [Authentication](/api/authentication) for rate limit details and how to monitor your usage.

## Getting Started

1. **Connect**: Add MidOS to your MCP client configuration
2. **Handshake**: Call `agent_handshake()` to personalize your setup
3. **Search**: Use `semantic_search()` or `search_knowledge()` to explore
4. **Learn**: Access skills with `list_skills()` and `get_skill()`
5. **Store**: Share learnings back with `episodic_store()` (Pro tier)
